Welcome to Social Spreadsheet. Social Spreadsheet allows editing of a spreadsheet by multiple users over a network. Here I will provide a few short notes on how it may be used.

To Begin

The File menu contains four options, Create(New), Join(Open), Save(Save), and Leave(Close). Select Create or Join to begin working on a spreadsheet. Create is to create a new spreadsheet to work on. Join is to begin working on an existing spreadsheet. For either selection a window will show asking for a server, port, filename, or password. If you have selected Create the filename and password are chosen by you.

To Edit A Cell

Select a cell by clicking on it. The cell's name will appear in the box labeled Cell, its value will appear in the box labeled Value, and its content will appear in the box labeled Content.

Edit a cell by typing in the Content box and pressing the enter or return key to finalize a change. Start your entry with the = character to enter a formula.

Select Save to save a file in its current location and under its current name. Select Leave to close your current window.

Formulas

In addition to numbers and strings, you may enter formulas that will calculate a value based on the values of other cells. Begin a formula by typing = in the content box. Allowable symbols in formulas are numbers, *, /, +, -, (, ), and variables. Variables are references to other cells. Allowable variables run from A1 to Z99.
